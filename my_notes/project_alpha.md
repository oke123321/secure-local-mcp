# Dự án Alpha: CI/CD Pipeline Automation

## Tổng quan
Dự án Alpha tập trung vào việc tự động hóa quy trình CI/CD bằng Python và Docker.

## Tech Stack
- Language: Python 3.11
- Containerization: Docker + Kubernetes
- CI/CD: GitHub Actions

## Trạng thái
Đang phát triển. Deadline: Tuần 30.

## Ghi chú kỹ thuật
- Cần review kiến trúc MCP Server trước khi merge vào main.
- SQLite FTS5 đã được kiểm chứng, tốc độ tìm kiếm O(log N) hiệu quả.
